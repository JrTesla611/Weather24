# 

A simple and reliable weather application that provides accurate forecasts using [Open-Meteo](https://open-meteo.com/) data. View current temperature, wind speed, and weather conditions for any city. Designed for speed, clarity, and privacy â€” no personal data is stored or shared.


## Features

- Open-Meteo data usage
- Live data 
- Easy to use SearchBox
- Clear Chart


## Installation

Install Weather24 directly from GitHub

```bash
  git clone https://github.com/JrTesla611/Weather24.git
  cd Weather24
```
Then open index.html in your web browser
